    Flood Mapping Using Machine Learning at NeurIPS 2025

    Flood Mapping Tutorial (Climate Change AI)

    Date: December 07, 2025
    Venue: Climate Change AI workshop at NeurIPS 2025, San Diego Convention Center, San Diego, CA

    I will be conducting a hands-on tutorial titled Flood mapping with optical and microwave satellite data: from indices to machine learning at NeurIPS 2025 as part of the Climate Change AI Tackling Climate Change with Machine Learning track.

    The tutorial focuses on practical approaches for mapping floods using MODIS, Sentinel-2, and Sentinel-1. We move from simple index-based and anomaly-based methods to scalable machine learning approaches designed for global application.

    Full tutorial details are available here:
    https://www.climatechange.ai/papers/neurips2025/120

    Global Flood Mapper (GFM)

    The tutorial is closely aligned with my work on GFM, an end-to-end framework for global flood extent mapping using satellite observations. Interface of the GFM:

    The published version of the Global Flood Mapper (GFM) focuses on automated processing of Sentinel-1 SAR data to generate consistent flood extent maps across diverse geographic regions and flood events. The corresponding peer-reviewed paper is available here.

    Global Flood Mapper v2 extends the original framework beyond flood extent mapping to support impact-oriented analysis.

    • Population exposure assessment using gridded population datasets
    • Flood depth estimation using elevation-based modeling
    • Event- and region-level impact summaries
    • Scalable, cloud-based workflows for rapid deployment

    Code, documentation, and examples for Global Flood Mapper and Global Flood Mapper v2 are available on GitHub:
    https://github.com/PratyushTripathy/global_flood_mapper
